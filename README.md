# Morphe Moment
[![CI](https://github.com/Seremegu/MorphMoment/actions/workflows/ci.yml/badge.svg?event=schedule)](https://github.com/elohim-etz/morphe-builder/actions/workflows/ci.yml)

Morphe Builder with my preferred settings.

Get the [latest CI release](https://github.com/Seremegu/MorpheMoment/releases).

## Building locally
### On Termux
```console
bash <(curl -sSf https://raw.githubusercontent.com/Seremegu/MorpheMoment/main/build-termux.sh)
```

### On Linux
```console
$ git clone https://github.com/elohim-etz/morphe-builder
$ cd morphe-builder
$ ./build.sh
```
